A compact multi-agent orchestration and evaluation framework derived from a phi-harmonic AI system. Focused on research, reasoning, and governance-friendly AI protocols.

## What It Does

- Runs structured tests on AI agents to observe behavior before deployment.
- Implements phi-harmonic modal orchestration patterns.
- Integrates OWASP Agentic Top 10 threat model awareness.
- Exposes governance-friendly audit and evaluation hooks.

## Quick Start

```bash
git clone https://github.com/Flickerflash/junior-apogee-app.git
cd junior-apogee-app
pip install -r requirements.txt
python app.py
```

## Evaluation

This project is evaluated using the **Goldcanstaytoday** benchmark:
- **Protocol:** 3-layer audit (Performance, Reciprocity, Lavender/Apogee)
- **Dataset:** Resume Apex Dataset 1
- **Target confidence:** 95% (k-fold CV)
- **Eval repo:** [Flickerflash/resumeapex-eval](https://github.com/Flickerflash/resumeapex-eval)

## Tech Stack

- **Python 3.11+** — core orchestration
- **Phi-calculus** — mathematical foundation for agent reasoning
- **OWASP Agentic Top 10** — threat model integration
- **GitHub Actions CI** — automated lint/test/build
